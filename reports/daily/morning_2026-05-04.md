## A8 Document Window Prep

Implemented opt-in consecutive-sentence context windows for CODE-ACCORD:

- `--doc-window-size N` joins up to N consecutive sentences from the same source
  document.
- `--doc-window-stride` controls sliding-window stride.
- Entity and relation word spans are offset into the joined context.
- CSV metadata document IDs are parsed from `metadata['ID']`.
- Original sentence order is preserved before windowing.

Validation:

- `xlm-roberta-base` and `bert-base-uncased` loader smoke tests passed with
  `doc_window_size=3`, `max_length=256`.
- A 3-step BERT training dry-run with `--doc-window-size 3` completed
  dev/test metric calculation without errors.
